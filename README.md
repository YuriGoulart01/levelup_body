# 🏋️ Level Up Body

O **Level Up Body** é uma aplicação web fullstack desenvolvida para acompanhamento da evolução física do usuário, permitindo o gerenciamento de dados corporais, monitoramento do progresso e visualização gráfica de métricas em tempo real. O sistema foi projetado para oferecer uma experiência intuitiva e responsiva, promovendo maior controle da evolução física de forma prática e organizada.

---

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido com o objetivo de aplicar conhecimentos em desenvolvimento FullStack, integração entre frontend e backend, autenticação segura, manipulação de dados e construção de interfaces modernas, seguindo boas práticas de arquitetura de software.

Além disso, o sistema busca proporcionar uma experiência personalizada para usuários que desejam acompanhar sua evolução corporal por meio de indicadores visuais e históricos de progresso.

---

## ✨ Funcionalidades

✅ Cadastro e autenticação de usuários  
✅ Login seguro com autenticação JWT  
✅ Login social com conta Google  
✅ Gerenciamento de perfil do usuário  
✅ Atualização de dados corporais  
✅ CRUD completo de informações  
✅ Visualização gráfica da evolução física em tempo real  
✅ Rotas protegidas  
✅ Interface responsiva e moderna  
✅ Integração completa entre frontend e backend
